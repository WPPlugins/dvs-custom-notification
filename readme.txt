A fully customizable notification/ call to action maker, enable any HTML/ Javascripts with 2 notification styles, resizeable and 9 placement option.

== Description ==
Custom Notification by DVS enables to write announcement/ notification or call to action to visitor using any HTML or javascript code. This plugin is fully customizeable, 2 notification types (styles), resizeable notification box, 9 placement options (compass) and super fast loads.

= Features =
* Super fast loads
* HTML or javascript code insertion enabled
* 2 Notification types (styles)
* Enable to be activated individually or both together
* 2 placements option (notification type 1)
* 9 placements option/ compass (notification type 2)
* Fixed position (floating on browser) 
* Dynamic Background color
* Resizeable Notification box
* Example included, easy and fully customize

== Installation ==
1. Download and unzip dvs-custom-notification.zip to your plugin folder.
2. Activate the plugin from the Plugins menu of your wordpress dashboard.
3. Go to the "Custom Notification" options in your dashboard check "Display Custom Notification", define widget placement, background color and and the content.
4. Done!
